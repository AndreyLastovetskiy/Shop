Структуры:

Shop:
  1) name - название магазина
  2) owner - владелец магазина
  3) allowedToCapture - промежуточный счет, при помощи, которого можно осуществлять возврат денег за купленный продукт
  4) exists - существует ли магазин
  
ShopProductList:
  1) shop - название магазина
  2) products - продукты в этом магазине и информация о них (цена, количество) по ID товара

ShopProduct:
  1) id - id продукта
  2) price - цена товара 
  3) amount - цена товара 
  4) exists - существует ли товар 

User:
  1) username - логин пользователя
  2) shop - владелец какого магазина
  3) passwordHash - пароль
  4) secretHash - доп. пароль для двухфакторной авторизации 
  5) maxRole - макс. доступная роль для пользователя
  6) currentRole - текущая роль пользователя 
  7) exists - существует ли пользователь

UserProducts:
  1) user - адрес пользователя
  2) shops - название магазина => информация о купленных продуктах

UserShopProducts:
  1) shop - название магазина
  2) products - id товара => информация о купленном продукте
  
UserOwnedProduct:
  1) id - id товара 
  2) amount - кол-во купленного товара 
  
ElevateRequest:
  1) sender - пользователь, который отправил заявку на повышение роли
  2) Role - роль, которую он требует для повышения
  3) shop - название магазина
  4) exists - существует ли заявка на повышение роли 
  
Product:
  1) id - id продукта 
  2) title - название товара 
  3) description - описание товара 
  
BuyRequest:
  1) author - автор заявки на покупку товара
  2) productId - id продукта, который мы хотим купить 
  3) amount - кол-во товара 
  4) shop - магазин, где мы будем покупать 
  5) accepted - одобрена ли заявка на покупку товара 
  6) reviewed - просмотрена ли данння заявка на покупку товара 
  7) exists - существует ли заявка 
  
RefundRequest:
  1) id - id заявки 
  2) buyRequestId - id товара, который мы хотим вернуть 
  3) author - автор заявки 
  4) accepted - одобрена ли заявка на покупку товара 
  5) reviewed - просмотрена ли данння заявка на покупку товара 
  6) exists - существует ли заявка
